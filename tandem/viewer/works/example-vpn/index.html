Возможности

Создан быть неудержимым

Ни одна VPN-сигнатура не попадает в поток данных. Провайдер видит обычный HTTPS-трафик к CDN.

Обход ТСПУ / DPI
Аппаратные DPI-комплексы у каждого российского провайдера не обнаруживают ни одной VPN-сигнатуры.
Невидим для провайдера
Трафик имитирует TLS-запросы к крупным CDN. Провайдер видит безобидный HTTPS-поток.
Нулевой оверхед
Протокол на базе QUIC — менее 5 мс задержки. 4K-стриминг без буферизации.
AES-256-GCM
Военный стандарт + ChaCha20 для мобильных. Полная прямая секретность — каждая сессия независима.
No-logs политика
Архитектура физически не позволяет хранить логи. Ni IP, ни DNS, ни timestamp.
Open Source
Полностью открытый код на GitHub. Соберите сами, проверьте — никаких бэкдоров и телеметрии.
Архитектура

Обход ТСПУ

ТСПУ — аппаратные DPI-комплексы у каждого провайдера. Блокируют по сигнатурам протоколов в реальном времени.

ArcVPN применяет стеганографическую маскировку — ни одна VPN-сигнатура не попадает в поток, ТСПУ пропускает трафик как обычный HTTPS.
